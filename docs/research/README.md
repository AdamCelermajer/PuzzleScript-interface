# Research Docs

This folder holds supporting research, prompt-spec, and reference materials used to explain or explore the project.

These files are not part of the main runtime path for the Node.js PuzzleScript service, the FastAPI adapter, or the Python client entry points.

- `context.acdl` and `main_loop.acdl` capture prompt and orchestration materials.
- `notebookLM/` contains supporting NotebookLM research assets.
