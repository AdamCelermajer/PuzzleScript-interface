# LIVE Framework Study

This folder owns the active LIVE-style Sokoban/framework study. It uses the local PuzzleScript ARC service and learns readable action-effect rules from neutral percepts.

## Command

```bash
python -m studies.LIVE_framework.run --help
```

The experiment targets `ps_sokoban_basic-v1` by default and writes generated outputs under `studies/LIVE_framework/output/`.
